# Carbon Launcher

> A minimal Fortnite launcher made for Carbon.

## Bun.sh or NodeJS

Launching the project can be done via [Bun.sh](https://bun.sh) or NodeJS.

- Upon installing Bun, run:
  ```bash
  bun install
  ```
  Once finished, run:
  ```bash
  bun run tauri dev
  ```

- For NodeJS:
  ```bash
  npm install
  ```
  Then run:
  ```bash
  npm run tauri dev
  ```
